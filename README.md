Necessary structures for storing information about music CDs in a collection, such as:

Artist/Group/Band Name
Album Name
CD Release Date
CD Price
Producer/Record Label Name
Musical Genre Code and Name
Total Number of Tracks on a disc, and the Number, Title, and Duration of each CD Track
Something that identifies whether the CD is a single (one CD), double (two CDs), or box set (more than two CDs)

An algorithm that allows filling in the information of this CD collection and performs some operations on it, such as:

Search by Artist/Group/Band
Search by Album
Search by Genre
Search by Track Title
List (print) the entire collection (Artist, Album, and Tracks)
